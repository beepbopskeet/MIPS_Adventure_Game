# 👑 The King of the Four Realms

A story-driven adventure game developed entirely in MIPS Assembly Language using the MARS simulator.

Created as a COMP 206 Term Project.

---

## 🎮 Game Overview

You are the ruler of a kingdom divided into four competing realms:

* Vampires
* Miners
* Priests
* Lithari

Every decision affects the balance of power between these factions.

Your choices influence:

* Hunger
* Strength
* Aggression
* Influence
* Faith
* Crystal Power
* Health

The fate of the kingdom depends on how you manage these competing interests.

---

## 🎯 Gameplay Mechanics

Players are presented with story events and dialogue choices.

Each decision changes faction statistics and may unlock:

* New story paths
* Hidden events
* Different alliances
* Alternative endings

No two playthroughs are guaranteed to end the same way.

## 🏁 Endings

The game contains multiple possible endings depending on:

* Realm support

## 🖥 IMPORTANT: Screen Setup Before Running

Before running the game in MARS:

### 1. Open Bitmap Display

Tools → Bitmap Display

### 2. Configure Display

Use the following settings:

```text
Unit Width: 1
Unit Height: 1

Display Width: 128
Display Height: 128

Base Address:
0x10010000
```

### 3. Connect Display

Press:

```text
Connect to MIPS
```

### 4. Assemble and Run

```text
Assemble (F3)
Run (F5)
```

The game will not display correctly if the Bitmap Display is not configured before execution.
